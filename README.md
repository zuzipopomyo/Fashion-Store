# Fashion Store

A simple fashion e-commerce frontend built with React, TypeScript, Tailwind CSS, and Zustand.

Live demo: [Fashion Store](https://zuzipopomyo.github.io/Fashion-Store)

## Features

- Browse products from Fake Store API
- Filter products by category and price
- View product details and choose quantity
- Manage a persistent shopping cart with Zustand
- Complete a multi-step checkout flow

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS
- React Router
- Zustand
- Axios

## Getting Started

```bash
npm install
npm start
```

Open `http://localhost:3000` in your browser.

## Scripts

- `npm start` - Start the development server
- `npm run build` - Create a production build
- `npm test` - Run tests
- `npm run deploy` - Deploy to GitHub Pages

## Data Source

This project uses [Fake Store API](https://fakestoreapi.com/) for product and cart data.
